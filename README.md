# Guia de Qualidade de Software: Comportamento, Agilidade e Testes

Este repositório documenta o estudo organizado no NotebookLM com foco em qualidade de software (QA): comportamento profissional, práticas ágeis e estratégias de testes.

- Link do estudo no NotebookLM:
	- https://notebooklm.google.com/notebook/0762ad7a-b42e-4a45-9403-ac2e532d2c81

Fontes (selecionadas dentro do NotebookLM — visíveis na captura):
- 10 Habilidades de Comportamento
- 5 exemplos de modelo de caso de uso
- Bug Report: o que é, pra que serve
- Junielton-lara / Roadmap-QA
- Metodologias Ágeis: Introdução (Scrum, Kanban)
- Selenium para iniciantes: tutorial
- Teste de software — Wikipédia
- Teste funcional — Wikipédia
- Tutorial do Postman: Como Usar

Recursos de estudo oferecidos pelo NotebookLM (painel "Estúdio"):
- Resumo em Áudio
- Resumo em Vídeo
- Apresentação gerada automaticamente
- Mapa mental
- Relatórios
- Teste (quiz)
- Cartões didáticos
- Infográfico (Beta)
- Tabela de dados
